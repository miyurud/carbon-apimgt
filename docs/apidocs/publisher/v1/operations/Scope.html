scope

Check given scope name is already exist

HEAD https://apis.wso2.com/api/am/publisher/v1.0/scopes/{name}

Using this operation, user can check a given scope name exists or not.

OAuth 2.0 Scope

apim:api_create apim:api_publish

Examples

Parameters

Type Name Description Schema Default
Path

name

required

Scope name

String

Responses

HTTP Code Description Schema

200

OK. Requested scope name exists.

404

Not Found. Requested scope name does not exist.
