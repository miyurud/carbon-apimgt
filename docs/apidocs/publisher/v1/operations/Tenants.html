Tenants

Check whether the given tenant already exists

HEAD https://apis.wso2.com/api/am/publisher/v1.0/tenants/{tenantDomain}

Using this operation, user can check whether a given tenant exists or not.

OAuth 2.0 Scope

apim:api_view

Examples

Get tenants by state

Request
GET https://localhost:9443/api/am/publisher/v1.0/tenants?state=active HTTP 1/1 Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8
Response
HTTP 1/1 200 OK Content-Type: application/json { "count" : 3, "list" : [ { "domain" : "test.com", "status" : "ACTIVE" }, { "domain" : "carbon.super", "status" : "ACTIVE" }, { "domain" : "test2.com", "status" : "ACTIVE" } ], "pagination" : { "offset" : 0, "limit" : 25, "total" : 3 } }
